# Project History

## Introduction

The Job Application Tracker was developed as a practical Microsoft Excel solution to help job seekers organize, manage, and monitor their job application journey. The idea behind the project was inspired by the challenges many applicants face when applying for multiple roles, including losing track of applications, forgetting important dates, and struggling to measure their job search progress.

The goal was to create a simple but professional tool that allows users to maintain a centralized record of their applications while gaining useful insights through an interactive dashboard.

## Project Planning

The project started with identifying the key information a job seeker needs to track throughout the recruitment process. The tracker was designed around important application details such as company information, job roles, application dates, recruitment stages, and progress status.

The focus was not only on data collection but also on creating a user-friendly experience. The workbook was structured to guide users from introduction to data entry and finally to performance analysis.

## Workbook Design

The workbook was designed with multiple worksheets, each serving a specific purpose:

- Welcome Page: Created as the landing page of the workbook, providing an introduction, key features, user guidance, and navigation.
- Job Tracker: Designed as the central data entry area where users record and manage their job applications.
- Dashboard: Developed to transform application data into visual insights, helping users monitor progress and understand their job search performance.
- Lists: Created to support consistent data entry through dropdown selections.

The structure was developed to make the workbook easy to navigate while maintaining a professional user experience.

## Development Process

During development, several features were introduced to improve usability and functionality.

The project included:

- A structured job application database.
- Dropdown lists for consistent data input.
- Automated calculations for dashboard metrics.
- Visual reporting through charts and summary indicators.
- A dedicated Welcome page to improve user onboarding.
- Navigation elements to make movement between worksheets easier.

The dashboard was designed to help users quickly understand important job search information, including application progress and recruitment activity.

## Testing and Improvements

Throughout the development process, different workbook features were tested to ensure that functionality was maintained.

One improvement considered was protecting formula cells to prevent accidental changes. However, testing showed that locking the formula columns affected the workbook's functionality. As a result, formula protection was not included in the current version to ensure users have a smooth experience.

The decision reflects the importance of balancing workbook protection with usability.

## Current Version

The current version represents a complete Job Application Tracker solution designed for personal productivity and professional portfolio demonstration.

The project showcases practical Microsoft Excel skills, including:

- Workbook design.
- Dashboard development.
- Data organization.
- Data validation.
- Formula implementation.
- User-focused interface design.
